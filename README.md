# Solvea Outreach Dashboard

Real-time dashboard for Solvea's outbound sales, Voice SDR, email tracking, and PR media operations.

## Live URL
https://mguozhen.github.io/solvea-dashboard/

## Features
- Voice SDR: calls, connect rate, SMS follow-ups
- Email funnel: sent → opened → clicked → replied → booked
- Hot leads: who opened/clicked with contact info
- PR media: pitched vs target database coverage
- Auto-refreshes every 60 seconds
